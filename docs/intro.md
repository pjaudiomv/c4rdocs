# 12 Step Meeting List — Knowledge Base

A searchable archive of community questions and answers for the
[12 Step Meeting List](https://github.com/code4recovery/12-step-meeting-list)
WordPress plugin, compiled from GitHub
[discussions](https://github.com/code4recovery/12-step-meeting-list/discussions)
and [issues](https://github.com/code4recovery/12-step-meeting-list/issues).

## How to use this site

- **Searching?** Hit <kbd>⌘</kbd>+<kbd>K</kbd> (or <kbd>Ctrl</kbd>+<kbd>K</kbd>) and type what
  you’re looking for — the search indexes every question and answer.
- **Browsing?** Use the sidebar. Questions are grouped by topic under **FAQ**.
- **Looking for code?** Check **Code Recipes** for copy-pasteable snippets
  pulled from answered questions.
- **Need the top hits?** See [Top Questions](./top-questions) for the
  most-upvoted, most-resolved Q&A across the entire community archive.

## About the content

Every page here links back to the original GitHub thread — click the
source link on any answer to see the full discussion, thank the
author, or follow up if something’s still unclear. See
[Sources & Credits](./sources) for more about how this archive is built.

Nothing here replaces the official plugin — for installation, configuration,
and the shortcode reference, start at the
[plugin README on GitHub](https://github.com/code4recovery/12-step-meeting-list#readme).
